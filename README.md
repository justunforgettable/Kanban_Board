# Kanban Board

A simple and interactive **Kanban Board** built using **HTML, CSS, and JavaScript** to help users organize and track tasks visually through different workflow stages.

## Features

* Create and manage tasks
* Drag-and-drop functionality
* Multiple workflow columns (To Do, In Progress, Done)
* Responsive and user-friendly interface
* Real-time task status updates

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
